Install a theme with demo package

This tutorial is a step-by-step guide that will show you how to install a Theme Demo Installation Package.

Download the Theme Demo Installation Package

  1. Go to our Website, choose a theme and download the Demo Installation Package.
  2. Extract the .zip file.
  3. Upload or copy the extracted folder to your webserver.
  4. Now just open the url to the uploaded folder and follow the regular installation instructions for Joomla or WordPress.

For Joomla users

On the last installation step don't forget to press the Install Sample Data button.
