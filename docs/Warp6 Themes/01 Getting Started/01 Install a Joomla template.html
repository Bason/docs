Install a Joomla template

In this tutorial we'll show you, how to install a template in your Joomla installation.

Install

To install a template based on the Warp framework use the extensions installer in the Joomla administration. Go to Extensions » Install/Uninstall.

Click on Browse..., select the downloaded template file and click in Upload & Install.

Activate

To activate the new template go to Extensions » Template Manager, choose the installed template and click on Make Default.

By clicking on the templates name, you'll be able to change all {doc: theme-settings text: template specific settings}.

When the installation does not work this way you may try to install the template this way:
  1. Download the {route: download-themes text: latest template version} and unpack all template files to a directory on your pc.
  2. Using FTP, upload this directory to the /tmp directory of your Joomla installation on your webserver.
  3. Go to the extensions installer in the Joomla administration, use the "Install from Directory" option to set the directory of your uploaded template files.
  4. Click on the Install button and Joomla will install it from the given directory.
