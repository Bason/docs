Create a new Style

You're able to add your own individual styles for a widget, if you want to customize the appearance.

Lets say you want to add a custom style to the slideshow widget. Just go to the folder /widgets/slideshow/styles, copy an existing style you want to customize and give it a unique name.

In the admin area you should be able to select your new created style.

Take a closer look at template.php, style.css and config.xml in your new style folder.

  • template.php: generates the output of the widget.
  • style.css: defines custom styles for the output.
  • config.xml: contains custom configuration settings.
