Menus

Manage and organize your menus with additional settings.

In Warp you have the possibility to enhance the mega menu and the dropdown menu with additional settings. Menu items can be filtered by the position to which there are assigned.

Menu Settings

The following table shows you all additional menu settings.

Setting Description
Columns Display up to 4 columns within the dropdown.
Width Enter a pixel value to set the width of the dropdown.
Icon Enter an icon class to set an icon to the menu item. Additionally the overview also allows you to set an icon to the according dropdown menu items.
