Update

Learn how to update your Warp theme without losing your customizations.

If a newer version of your theme or Warp is available, you will receive a notification in the Warp administration. In that case you need to visit the Downloads section of your theme and then download the latest theme package.

Update Warp

  1. In the theme directory on your server you will find a folder named warp.
  2. Rename this folder to warp_1, or something similar and save a copy of it. That way, you will be able to reuse the content in case the update fails.
  3. Download our free Master Theme and unzip the package.
  4. Copy the warp folder from the Master Theme and replace the one in your current installation with this one.

Update Theme

The file verification in the Warp administration shows all modified and missing files in your theme to help you detect which files you have customized before you start updating. To update your theme the follow these steps.

  1. First of all make a backup of all your customized theme files, so you can recover all modifications at any time.
  2. Download the latest theme version and extract the package to your Desktop.
  3. Copy all files to /templates/yoo_MY-THEME for Joomla or /wp-content/themes/yoo_MY-THEME for WordPress and replace all files.
  4. Recover your modifications by merging your customized files with the new ones.

IMPORTANT We strongly recommend to use the /custom folder for all modifications and make a backup before updating. That way all your changes will be kept when updating your original theme files by replacing them.
