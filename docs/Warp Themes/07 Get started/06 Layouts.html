Layouts

The layout settings manage the positioning of different layout parts.

This section makes the arrangement of your content very easy and flexible. You can enable and disable system output and define the layout of the theme's widget or module positions. Additionally you have the possibility to store your modifications in a layout profile and assign it to different pages.

Content

This option allows you to hide the main system output. This is useful, if you want to position only widgets/modules on certain pages, without leaving redundant space.

Sidebars

The Warp framework provides the following layout options for the sidebars.

Setting Description
Width Choose from the provided percent values to set the sidebar width.
Alignment Define, whether the sidebar is published on the left or right side of the content.

The sidebar settings allow you to create a fully functional multi-column layout. The only requirement is to publish a widget or module on a sidebar position to display the selected layout.

Grid

This table gives you an overview of the grid layout settings.

Setting Description
Layout Define, whether the widgets or modules are positioned side-by-side, stacked vertically or first doubled.
Divider Add a vertical divider between parallel or first doubled widgets/modules to seperate them from each other.
Responsive Check this option so that widgets or modules take up full width and stack vertically in narrow viewports. Otherwise they will preserve the original layout.

For more information on widgets and modules, take a look at the Widgets & Modules tutorial.

Layout Manager

The layout manager allows you to create new layouts and assign them to different pages. Warp provides a default layout as a reference.

How to create a new layout?

To create a new layout, please follow these steps:

  • 1. Hit Add and enter a name to create a layout.
  • 2. That's it! You can start customizing your layout.
  • 3. Finally, assign your layout to the appropriate pages within the scrollable box at the bottom of the layout setting page.

IMPORTANT You can't assign two different layouts to one page.
