/css

Contains the ie8.css file.

File Description
ie8.css Will fix basic issues for the Microsoft Internet Explorer 8.

/custom

This is the place, where you can store all your {doc: customization text: custom markup and additional files}.

/images

The /images folder contains all images used in the theme.

/js

All theme specific JavaScript can be found in this folder.

File Description
theme.js Responsible for all JavaScript effects and all JavaScript based functions.

/layouts

This folder contains the core files responsible for the theme's layout.

File Description
theme.config.php Includes layout calculations and assembles the theme's CSS and JS files. For more information take a look at {doc: assets text: JS, CSS and LESS}.
theme.php Provides the complete HTML markup for the base theme layout. For more information take a look at {doc: theme-layout text: Theme layout}.
widget.php Responsible for displaying all widget variations. For more information take a look at {doc: widgets-2 text: Widgets & Modules}.

/less

All LESS files of your theme are stored in this folder. Changes to these files will be compiled into CSS and saved in the /styles/STYLE-NAME/css folder.

File Description
/bootstrap Contains all Bootstrap related LESS files.
/uikit Contains all UIkit related LESS files, the frontend framework on which Warp is built.
bootstrap.less Joomla! only: Imports all Bootstrap components and related files. For more information, take a look at the Bootstrap website.
customizer.json Defines which customizer parameters will be displayed in default or advanced mode. It groups UIkit variables, sets what can be defined through the color picker, direct input or a select box and adds fonts.
theme.less Provides the theme's markup and defines its styles and compiles the /uikit/uikit.less file within the theme.css file. You can also add your own LESS here.

/styles

Here you can find all available style variations with the compiled CSS files. You don't actually work in this folder. Changes made in one of the LESS files will automatically be compiled into CSS and saved in this folder.

/warp

This folder contains the actual Warp 7 core framework.

Root files

This is a little overview of some files you should keep in mind.

File Description
changelog.php Gives you information on the version number of your Warp theme and all changes that have so far been made.
config.default.json Stores all theme related default backend parameters.
config.json Stores all your saved theme related backend parameters. This file will be generated automatically as soon as you save your settings.
config.php Registers all LESS files for compilation. For more information, take a look at {doc: assets text: JS, CSS and LESS}.
config.xml This file contains the backend parameters of your theme. For more information, take a look at {doc: config text: Config.xml}.
templateDetails.xml (Joomla)
theme.xml (WordPress)		 This file contains general information on your theme, like the name, release date, directories and module settings.
warp.php This file loads the Warp Framework.
favicon.ico Will be displayed in your browser bar. To change the favicon, simply replace that file.
apple_touch_icon.png Will link to your website, if you save it to the home screen of your mobile device. To change the icon, simply replace that file.
