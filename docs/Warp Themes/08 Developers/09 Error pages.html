Error pages

The error pages available in Warp themes are white-label, minimalist designs that serve as an override to the system defaults.

File Description
error.php In case of a missing page, the error page layout will be displayed with a 404 error title and description text. The file is located in warp/layouts/
offline.php In the event you need to perform maintenance on your website, activating the offline mode page will notify your users that your site is currently unavailable. The Offline mode page is only available for Joomla. The file is located in warp/systems/joomla/layouts/

NOTE To override the HTML output, check the Customization and System overrides docs. You can find more information on the language output in the Language chapter.
