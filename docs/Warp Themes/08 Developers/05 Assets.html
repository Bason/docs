JS, CSS and LESS

Add JavaScript, CSS or LESS files to your Warp based theme.

The configuration file layouts/theme.config.php initializes all PHP classes and loads the necessary CSS and JavaScript. If you need to load custom CSS and JavaScript files, this is the place to do it. Enabled compression and Data URI will automatically be applied to all files you add here. Find out more about overriding theme files with your custom CSS or JavaScript in the {doc: customization text: Customization} doc.

Add a JavaScript file

If you want to add your own script, simply copy it to the js/ directory of the theme and add the following to the theme.config.php configuration file, so the file will automatically be added to the template header.

// add javascripts
$this['asset']->addFile('js', 'js:my.js');

Add a CSS file

To add your own custom CSS file, you can do so by using the following syntax in the theme.config.php, so the file will automatically be added to the template header.

// load css
$this['asset']->addFile('css', 'css:my.css');

Add a LESS file

The config.php registers all LESS files for compilation. When hitting Compile LESS in the Settings section of the Warp administration, these files will be compiled into CSS and saved in the theme's styles/STYLENAME/css/ folder. If you want to add your own LESS file to be compiled into CSS, you need to register it here as well.

'less' => array(

    'vars' => array(
        'less:theme.less'
    ),

    'files' => array(
        '/css/theme.css'     => 'less:theme.less',
        '/css/bootstrap.css  => 'less:bootstrap.less'
    )

),

Add a custom font

Warp gives you access to all fonts available in the Google fonts directory. But occasionally you might want to add your own font or one that is not provided by Google.

To do so, just create a folder, for example fonts and add your font files to it. Then add the @font-face rule for your fonts to less/theme.less and lastly add your font option and the correct path to the list inside less/customizer.json. And that's it! You can now select your font from within the customizer.
