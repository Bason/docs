The Changelog

The changelog is a physical file that contains a record of the changes made to Warp themes between each version. Each section will be denoted by the version number, followed by a list of changes, using the following legend:

* -> Security Fix
# -> Bug Fix
$ -> Language fix or change
+ -> Addition
^ -> Change
- -> Removed
! -> Note

Changelog Location

There are two changelogs: One for the theme itself and one for the Warp theme framework. The theme changelog CHANGELOG.md can be found in your theme directory. The Warp framework changelog can be found by visiting /warp/README.md within your theme directory.

While all efforts are made to keep the changelog up to date and accurate, it should be considered that non-exhaustive and proper file comparison tools should be used to locate any modifications that may conflict with your installation.
