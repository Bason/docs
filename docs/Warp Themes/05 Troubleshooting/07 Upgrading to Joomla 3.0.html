Upgrading from Joomla 2.5 to Joomla 3.0

Please follow the following guide.

Updating Joomla

Please log in to your Joomla 2.5 administration and click on Components -> Joomla! Update. Now click on Options on the top right and select Short Term Support for Update Server. Just click on Install the Update.

If no update was found, you need to need to purge the cache by going to Extensions -> Extension Manager -> Update and click on Purge Cache. Then just go back to Joomla! Update.

Updating the theme

To update the theme you need to download the latest Joomla 3.0 theme from our download directory and replace the theme folder of your current 2.5 theme /templates/yoo_templatename with the new one.

Pro tip: If you have already made heavy adaptions to your the theme, you can also just replace the current warp folder with the 3.0 theme warp folder.
