Update custom elements ZOO 2.4 » 2.5+

This short tutorial will show you how to update your custom elements from ZOO 2.4 to ZOO 2.5.

Updating the element class

The elements data storage has been changed from XML to JSON. Because of this the elements data handling has changed, the ElementData class has been removed. Instead of calling $this->_data->get('value');, we now call $this->get('value');.

The call to $this->config->get('value');, has changed to $this->get('value');

Some of the methods and properties have been made deprecated. You'll find them in the element.php file.
