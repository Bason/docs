Update templates ZOO 2.1 » 2.2+

This short tutorial will show you how to update your custom templates from ZOO 2.1 to ZOO 2.2.

Updating the templates

There have been a few changes to methods of the category object in ZOO 2.2.

Below are the two methods that have changed and that need to be replaced.

ZOO 2.1 ZOO 2.2
countItems() replace with » totalItemCount()
countChildrensItems() replace with » childrenHaveItems()

You can simply browse through the template files in your custom application or custom template and exchange the code. In the apps of the app bundle you usually find these lines of code in the files:

  • \media\zoo\applications\YOURAPP\templates\YOURTEMPLATE\_category.php
  • \media\zoo\applications\YOURAPP\templates\YOURTEMPLATE\_categories.php
  • \media\zoo\applications\YOURAPP\templates\YOURTEMPLATE\frontpage.php
  • \media\zoo\applications\YOURAPP\templates\YOURTEMPLATE\category.php

Example

Code that looked like this:

$item_count = ($this->params->get('template.show_sub_categories_item_count')) ? ' <span>('.$child->countItems().')</span>' : '';

should now look like this:

$item_count = ($this->params->get('template.show_sub_categories_item_count')) ? ' <span>('.$child->totalItemCount().')</span>' : '';
