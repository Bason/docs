Install The ZOO With Democontent

This tutorial is a step-by-step guide that will show you how to install the ZOO Demo Installation Package.

  1. Go to the ZOO download section, choose your Joomla version and download the Demo Installation Package.
  2. Extract the archive file.
  3. Upload or copy the extracted directory to your webserver.
  4. Now just open the url to the uploaded directory and follow the regular installation instructions for Joomla.

On the last installation step don't forget to press the Install Sample Data button.
