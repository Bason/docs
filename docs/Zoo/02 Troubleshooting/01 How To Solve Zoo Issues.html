How To Solve ZOO Issues

If you are having trouble or issues with the ZOO, this can have numerous causes. The ZOO has a build in system check, to detect some of the problems. But there are some commen issues that might arise:

JS errors

If you are having trouble with functions in the ZOO, this might be due to a jQuery conflict. Please read this tutorial on how to detect and resolve the problem.

No CSS being loaded

This might be due to permission issues. Please read this tutorial on how to detect and resolve the problem.

Blank Page

This is most likely due to a php error being thrown. Please enable the display_errors parameter in your php.ini. Only with an error message the problem can be found.
